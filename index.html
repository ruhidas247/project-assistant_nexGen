📋 GitHub Setup Guide (click to expand) ▼
Step 1 — Create a GitHub repo (can be private):
Go to github.com/new → name it e.g. project-assistant-data → click "Create repository"

Step 2 — Create a Personal Access Token:
Go to GitHub → Settings → Tokens (classic) → Generate new token
• Note: Project Assistant  • Expiration: No expiration (or 1 year)
• Check only: ✅ repo (full control of private repositories)
• Click "Generate token" → copy it immediately (shown only once!)

Step 3 — Fill in below:
• Token: paste the ghp_... token
• Repo: your-username/project-assistant-data

Step 4 — Deploy to GitHub Pages (to launch the app):
• In your repo → Settings → Pages → Source: Deploy from branch → main → /root
• Upload the HTML file to the repo root as index.html
• Your app will be live at: https://your-username.github.io/your-repo/
